# 🚀 Quick Start - Hot Reload

## Passo a passo rápido:

### 1️⃣ Inicie o desenvolvimento
```bash
npm run dev
```

### 2️⃣ Acesse no navegador
```
https://localhost:3000
```

### 3️⃣ Aceite o certificado
Quando o navegador avisar sobre certificado não seguro:
- Clique em **"Avançado"**
- Clique em **"Continuar para localhost (não seguro)"**

### ✅ Pronto!

Agora qualquer alteração em arquivos `.php` ou no Tailwind CSS vai recarregar a página automaticamente!

---

## 📝 Comandos úteis

- `npm run dev` - Inicia watch do Tailwind + Browser-Sync
- `npm run build:css` - Compila CSS para produção
- `npm run watch:css` - Apenas watch do Tailwind (sem hot reload)

---

## ⚠️ Lembre-se

- Use **HTTPS** (não HTTP): `https://localhost:3000`
- A porta pode variar (3000, 3001, 3002...) - verifique no terminal
- Aceite o certificado auto-assinado quando solicitado
